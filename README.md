# PointAndClickShooter

A point and click shooting game.

## Requirements

* Processing 2.x.x or 3.x.x

## Clone and Run

Clone into PointAndClickShooter directory.

```
$ git clone --branch develop <url/to/pointandclickshooter.git> PointAndClickShooter
```

After cloning, open and run with the Processing Development Environment.
